This is Droplings for Mac.

It's a menubar icon. If you drop a file on it, it'll build a nice preview page around it, upload both to your Dropbox, then copy the public preview link to your clipboard.

Droplings comes with a clean preview page theme (see the download link below for an example), but you can use your own custom themes. (There's documentation.)

The maximum file size of the files you share is only restricted by the size of your Dropbox account.

Droplings requires OS X 10.6 or higher, running on a 64bit architecture (i.e. any 2007+ Mac should do). And of course you need a free or Pro Dropbox account.

Droplings is free at the moment as the 0.9.x versions are considered betas. Once we reach 1.0 there'll be a (modest) price tag.
